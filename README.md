git init
(Isso avisa o computador: "A partir de agora, comece a vigiar as mudanças nessa pasta".)

git add .
(O ponto final é importante! Ele significa "Pegue todos os arquivos que eu criei e prepare eles para serem empacotados".)

git commit -m "meu primeiro commit 🎉"
(Isso é como fechar a caixa com uma fita adesiva e colocar uma etiqueta dizendo o que tem dentro.)

git branch -M main
(Garante que a linha do tempo principal do código vai se chamar 'main'.)

git remote add origin [COLE_AQUI_O_LINK_DO_GITHUB]
(Aqui ele pega aquele link .git da tela que deixamos aberta no Passo 1. Isso cria a ponte invisível entre o PC dele e a nuvem do GitHub.)

git push -u origin main
(O grande momento! É o caminhão de entrega levando a caixa do computador dele lá pro servidor do GitHub.)
